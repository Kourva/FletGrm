# Installation
1. **Clone repository**:
    ```bash
    git clone https://github.com/kozyol/FletGrm
    ```
2. **Install flet**:
    ```bash
    pip install flet
    ```
3. **Run the app**:
    ```bash
    flet run main.py
    ```
    Or using python:
    ```bash
    python main.py
    ```
